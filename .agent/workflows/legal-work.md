# Legal Work Workflow

Streamline IP tasks: patent applications, trademark searches, and contract review.

## Patent Drafting

### Step 1: Invention Disclosure

```markdown
## Invention Disclosure Form

### Title
[Descriptive title]

### Inventors
- Name, Citizenship, Employer

### Problem Statement
What problem does this solve?

### Prior Art
What exists? How is this different?

### Technical Description
How it works in detail.

### Key Innovations
1. Novel aspect #1
2. Novel aspect #2

### Potential Claims
- Method claims (steps)
- System claims (components)
- Apparatus claims (structure)
```

---

### Step 2: Prior Art Search

| Database | URL | Coverage |
|----------|-----|----------|
| **Google Patents** | [patents.google.com](https://patents.google.com) | Global, free |
| **USPTO** | [patft.uspto.gov](https://patft.uspto.gov) | US patents |
| **WIPO** | [patentscope.wipo.int](https://patentscope.wipo.int) | International |
| **Espacenet** | [espacenet.com](https://worldwide.espacenet.com) | European |

### Search Strategy
```
Keywords: "autonomous drone" AND "swarm"
CPC Codes: G05D1/10 (aircraft control)
Inventor: Known competitor inventors
Assignee: Competitor company names
```

---

### Step 3: Draft Application

#### Provisional Patent (Quick Protection)
- **Cost**: ~$320 (small entity)
- **Timeline**: 12 months to convert
- **Requirements**: Description + drawings

#### Non-Provisional Structure
```markdown
### Title
### Background
- Field of Invention
- Problems with Prior Art
### Summary
### Brief Description of Drawings
### Detailed Description
### Claims
1. Independent Claim (broadest)
   2. Dependent Claim
4. Independent Method Claim
```

---

### Step 4: File with USPTO

1. Account at [patentcenter.uspto.gov](https://patentcenter.uspto.gov)
2. Prepare: Application Data Sheet, Specification, Drawings, Declaration
3. Pay fees (~$1,600 small entity)
4. Receive filing receipt

## Trademark Registration

### Step 1: Clearance Search

| Resource | URL |
|----------|-----|
| **TESS** | [tmsearch.uspto.gov](https://tmsearch.uspto.gov) |
| **WIPO Global Brand** | [branddb.wipo.int](https://branddb.wipo.int) |
| **Domain Search** | WHOIS lookup |

### Search Strategy
```
Exact: "SPRINGROLL"
Phonetic: "SPRING ROLL"
Synonyms: "AUTUMNROLL"
```

---

### Step 2: File Application

At [teas.uspto.gov](https://teas.uspto.gov):

- [ ] Word mark OR design mark
- [ ] International Class (1-45)
- [ ] Goods/services description
- [ ] Specimens showing use
- [ ] Filing fee ($250-350/class)

## Contract Review Checklist

### Software License
- [ ] License scope (perpetual vs. term)
- [ ] Usage restrictions
- [ ] Warranty disclaimers
- [ ] Liability cap
- [ ] IP ownership

### Service Agreement
- [ ] Scope of work
- [ ] Payment terms
- [ ] Deliverables
- [ ] Termination provisions

### Employment/Contractor
- [ ] IP assignment clause
- [ ] Work for hire provisions
- [ ] Non-compete scope
