## How to Run

### Run Today's Schedule
```javascript
import { TeamOrchestrator } from './services/TeamOrchestrator';

// Get today's agenda
const agenda = TeamOrchestrator.getDailyAgenda();
console.log(agenda);

// Queue and run all daily tasks
TeamOrchestrator.queueDailyTasks();
await TeamOrchestrator.runDailySchedule();
```

### Run Specific Agent
```javascript
// Run Finance agent's tasks for today
await TeamOrchestrator.runAgentWeeklyTasks('finance');
```

### Check Status
```javascript
const status = TeamOrchestrator.getStatus();
console.log(status);
```
