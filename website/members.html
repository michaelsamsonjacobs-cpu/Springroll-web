üèÅ Getting Started

1

Download & Install

Download the Windows installer above and run it. Windows SmartScreen may warn you ‚Äî click "More info" then "Run anyway".

2

Sign In with Google

Use the same Google account you used to purchase access. Your email must be in our whitelist.

3

Connect a Workspace

Click the + button in the left sidebar to index a folder. This lets agents access your local files.

4

Start a Conversation

Type in the chat box to interact with AI. Use the sidebar to switch between agents (Ralph, Legal, Dev, etc.).

üéõÔ∏è Command Center

The Command Center is a visual canvas where you can orchestrate multiple agents simultaneously.

  • Drag agents onto the canvas to position them
  • Click an agent to view its terminal output
  • Connect agents by drawing lines between them to create workflows
  • Right-click to add new agents or delete existing ones

Agents run in isolated Git worktrees, so they can work in parallel without conflicts.

ü§ñ Ralph Agent

Ralph is your autonomous worker that learns from every task.

  • Navigate to Ralph Agent in the sidebar
  • Type a mission: e.g., "Write a project proposal for AI tutoring platform"
  • Select the task type (Document, Code, Research)
  • Click Start Mission
  • Watch Ralph break down the task, execute steps, and self-correct
  • After completion, click Save to Global Brain to help Ralph learn
